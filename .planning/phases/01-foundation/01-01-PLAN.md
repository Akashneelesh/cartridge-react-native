  <files>package.json</files>
  <action>

    ```bash
    npm install starknet@^9 @cartridge/controller
    ```

    - starknet: Provider, Contract, and account interactions

    Do NOT install @starknet-react packages yet - those are for web React hooks.
    We'll use starknet.js directly for simpler React Native integration.

  </verify>
  <done>starknet.js ^9.x and @cartridge/controller installed</done>
